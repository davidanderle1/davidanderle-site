# Systemic financial risk model — methodology sketch

This file is part of the public flagship build on davidanderle.com.

## Purpose
Create a visible, reproducible framework that explains how local weakness can become system-wide fragility once institutions are connected through shared exposures, funding channels, and delayed information.

## Current status
- The current CSV on the site is **illustrative synthetic data**
- It is included to show structure, variables, and scenario logic
- It should be replaced over time with real public-source inputs

## Variables in the sample file
- `capital_ratio`
- `short_term_funding_share`
- `network_centrality`
- stress outcomes across four scenarios
